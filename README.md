# mtuq
moment tensor (mt) estimation with uncertainty quantification (uq)


Install
-------

1) `git clone https://github.com/uafseismo/mtuq.git`

2) `cd mtuq`

3) `pip install -e .`


Run example
-----------

4) `./data/examples/unpack.bash`

5) `./examples/GridSearch.DoubleCouple.Serial.py`
